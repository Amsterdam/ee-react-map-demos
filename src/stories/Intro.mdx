# Gemeente Amsterdam Map Demos

This project aims to include demos and examples of common use-cases of maps for Gemeente Amsterdam. The preferred map library is [Leaflet](https://leafletjs.com/) implemented in the frontend library [React](https://react.dev/).

Each demo is offered in two flavours:

1. [React](https://react.dev/) and [Leaflet](https://leafletjs.com/) for those who prefer native JavaScript/TypeScript. *There are also some React and Leaflet examples that use <a href="https://react.dev/learn/passing-data-deeply-with-context" target="_blank">React's context</a>, which assists with larger/advanced maps features.*
2. [React](https://react.dev/) and [React-leaflet](https://react-leaflet.js.org/) for those who prefer JSX.

For each demo we also try to includes documentation with following structure:

- General description
- Background/useful to know caveats
- Overview of the files required to implement
- And the code from each file
