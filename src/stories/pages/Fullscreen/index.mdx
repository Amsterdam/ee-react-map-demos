# Amsterdam Fullscreen Page Demo

## Description

The container for the fullscreen page is set up like this:

```jsx
<AspectRatio ratio="wide">
  <Map mapOptions={{ scrollWheelZoom: false }}>
    <ZoomControl />
  </Map>
</AspectRatio>
```

The map component is a wrapper around the Leaflet map library and sets a context for child components. The `mapOptions` prop is passed to the Leaflet map instance.
