## Background

### Amsterdam GeoJSON/tile layer

GeoJSON is a widely-used format for encoding geographic data structures using JavaScript Object Notation (JSON). If data is available in GeoJSON format, it often provides a universal solution for mapping libraries. The React-Leafelt GeoJSON component accepts a GeoJSON object as data.
The markers of a React-Leaflet GeoJSON component can be styled by the pointToLayer
