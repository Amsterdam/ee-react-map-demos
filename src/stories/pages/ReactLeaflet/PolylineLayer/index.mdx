## Description

This is a polyline layer on a React-leaflet map. A polyline layer is used to display lines on a map. It can be configured, extended and restyled (<a href="https://leafletjs.com/reference.html#polyline" target="_blank">see docs</a>). You can create a Polyline object with multiple separate lines (MultiPolyline) by passing an array of arrays of geographic points.
