# Description

This is the Amsterdam base layer on a React-leaflet map. To accomplish this there are three files:
1. The React components
    * [BaseLayer.tsx](#1-baselayertsx)
2. The CSS styles (1 file)
    * [styles.module.css](#1-stylesmodulecss)
3. Utils (1 file)
    * [getCrsRd.ts](#1-getcrsrdts)
    - This util file handles the CRS (Coordinate Reference System) because Gemeente Amsterdam by default uses [Rijksdriehoeksco√∂rdinaten](https://www.kadaster.nl/zakelijk/registraties/basisregistraties/rijksdriehoeksmeting/rijksdriehoeksstelsel).
