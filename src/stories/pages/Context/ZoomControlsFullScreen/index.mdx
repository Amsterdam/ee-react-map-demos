# Context - Zoom Controls (Full screen)

**[GitHub Repo URL](https://github.com/Amsterdam/ee-react-map-demos/tree/main/src/pages/MapContextExample/ZoomControlsFullScreen)**

## Description

Using the [Amsterdam Design System](https://designsystem.amsterdam/) we implemented some custom buttons to act as the [Leaflet map zoom controls](https://leafletjs.com/reference.html#control-zoom) in a full webpage.

In this example, the Map context exports a single element - the Leaflet map object. This enables you to interact with your Leaflet map natively in any component (that is a child to the Map context provider component).
